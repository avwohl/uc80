Recent Fixes:
- Fixed compound literal of struct type used as a flat array element initializer
  (test 00216).  Pattern:
      struct Wrap local_wrap[] = {
          ((struct Wrap){inc_global}),
          inc_global,
      };
  _gen_flat_array_init was treating the (struct Wrap){...} compound literal
  as a flat sequence of struct members — the compound's address was stored
  into local_wrap[0].func instead of inc_global being copied out, so calling
  through local_wrap[0].func jumped into the compound's data and looped until
  cpmemu's instruction limit fired.  Fix: when flat-array elem is a struct
  and the value is a Compound matching that struct type, recurse with
  val.init.values as the member values for that one element.
- Fixed negative constant comparison bug in AST optimizer constant folding
- Fixed shift operator result type (C99 6.5.7 compliance)
- Fixed float function calls and sin() implementation
- Fixed enum switch cases and float operations
