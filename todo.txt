compilation modules by listing all of them on the command line.  Given
the target has to fit in a 64K address space this allows global optimzaiton
of the entire program.  Also, given the lack of good baseregister+offset
modes global analays can allow allocating static variables for routnine
parameters and locals when the routine is not recursive.  in our ../ucow
compiler this was used to great effect to make locals static.  Then when
it found routines that are not live at the same time the different routines
shared the static local space.
